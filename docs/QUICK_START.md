# 快速启动指南

## 🚀 快速开始

### 1. 启动后端服务器

```bash
cd backend
uv run python main.py
```

后端服务器将在 `ws://localhost:8765` 启动。

### 2. 启动前端（演示版本）

直接在浏览器中打开 `frontend/demo.html` 文件即可。

### 3. 测试游戏流程

1. 在浏览器中打开 `frontend/demo.html`
2. 输入玩家名称，点击"加入游戏"
3. 在另一个浏览器窗口中打开 `frontend/demo.html`
4. 输入不同的玩家名称，点击"加入游戏"
5. 在第三个浏览器窗口中打开 `frontend/demo.html`
6. 输入第三个玩家名称，点击"加入游戏"
7. 在任意窗口中点击"开始游戏"
8. 开始游戏！

## 📱 移动端测试

1. 确保后端服务器正在运行
2. 在手机浏览器中访问 `http://<你的电脑IP>:<端口>/demo.html`
3. 或者将 `demo.html` 文件部署到 Web 服务器

## 🔧 开发模式

### 启动 React 开发服务器

```bash
cd frontend
npm install
npm run dev
```

访问 http://localhost:3000

## 🐛 常见问题

### 问题 1：WebSocket 连接失败

**症状**：点击"加入游戏"后提示连接失败

**解决方法**：
1. 确保后端服务器正在运行
2. 检查端口 8765 是否被占用
3. 检查防火墙设置

### 问题 2：玩家列表不更新

**症状**：加入游戏后看不到其他玩家

**解决方法**：
1. 刷新浏览器页面
2. 检查浏览器控制台错误
3. 重启后端服务器

### 问题 3：出卡失败

**症状**：点击出卡按钮没有反应

**解决方法**：
1. 确保游戏已开始
2. 确保是当前回合
3. 检查卡牌是否可出

## 🎮 游戏流程

1. **加入游戏**
   - 输入玩家名称
   - 点击"加入游戏"
   - 等待其他玩家

2. **开始游戏**
   - 当有 3 名或更多玩家时
   - 点击"开始游戏"
   - 进入游戏界面

3. **出卡**
   - 点击自己的卡牌
   - 选择出卡方式（调和、质疑、特技）
   - 等待其他玩家出卡

4. **游戏结束**
   - 当满足胜利条件时
   - 显示获胜玩家
   - 可以重新开始

## 📝 注意事项

1. **后端必须先启动**：前端需要连接到后端 WebSocket 服务器
2. **至少 3 名玩家**：游戏需要至少 3 名玩家才能开始
3. **网络连接**：确保前后端在同一网络或可访问
4. **浏览器兼容性**：推荐使用现代浏览器（Chrome、Firefox、Safari）
